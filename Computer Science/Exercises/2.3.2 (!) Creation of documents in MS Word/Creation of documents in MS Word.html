Задание "Создание документов в MS Word - 2"

 

  1. Откройте файл Невербальный способ общения.docx (часть курсового проекта).
  2. Измените поля для страниц документа:
    1. левое - 3 см,
    2. правое - 1,5 см,
    3. нижнее 2 см,
    4. верхнее - 2 см
  3. Осуществите проверку правописания (Рецензирование \ Правописание).
  4. Некоторые заголовки первого и второго уровня уже оформлены стилями Заголовок 1 и Заголовок 2. Примените данные стили к остальным заголовкам (на данный момент они выделены красным и зеленым цветами).
  5. Измените стили заголовков:
    1. Заголовок 1: цвет - любой, шрифт - Calibri, кегль - 18пт, выравнивание - по центру, интервал перед абзацем - 30пт.
    2. Заголовок 2: цвет - любой (на тон светлее Заголовка 1), шрифт - Calibri, кегль - 14пт, выравнивание - по центру.
  6. Для перемещения по документу перейдите в режим Область навигации (Вид \ Область навигации).
  7. Каждый Заголовок первого уровня (Ладони, Жесты рук и кистей и т.д.) переместите на следующую страницу в качестве нового раздела (Разметка страницы \ Разрывы \ Следующая страница).
  8. В тексте под заголовками «Другие известные жесты и движения» и «Сигналы глаз» рисунки не имеют названий. Вставьте названия для четырех рисунков в приведенной последовательности начиная с первого (Ссылки \ Вставить название):
    1. «Любитель сидеть верхом на стуле»
    2. «Готов к действиям»
    3. «Готовность закончить встречу или разговор: руки на коленях»
    4. «Использование ручки для контролирования взгляда собеседника».
  9. Примените сжатие ко всем рисункам одновременно (96 пикселей на дюйм).
  10. Вставьте сноски (положение - внизу страницы, формат номера - 1,2,3) с помощью команды Ссылки \ Вставить сноску:
    1. К первому абзацу «Что в сердце варится, на лице не утаится» с текстом «Русская народная пословица»
    2. К четвертому абзацу «Интуиция - чутье...» с текстом «Выдержка из словаря Ожегова С.И.»
    3. К фамилии «Ниренбергом» с текстом «Вместе с Калеро является автором книги «Как читать человека словно книгу. Язык тела»
  11. В разделе под заголовком «Защитный барьер, сформированный с помощью ног» найдите текст, шрифт которого имеет оранжевый цвет. Для данных абзацев создайте пользовательский нумерованный список - Стадия 1, Стадия 2 и т.д.:

                       

12.  В начале документа создайте новый раздел документа. В новом разделе вставьте Оглавление (Ссылки \ Оглавление \ Оглавление).

13.  Перейдите в конец документа и создайте новый раздел. В новом разделе вставьте  Список терминов. Этапы создания списка терминов:

    1. Выделите текст, для которого создается термин
    2. Вкладка Ссылки \ Пометить элемент (Alt+Shift+X)
    3. Настроить необходимые параметры

4. Кнопка Пометить

5. Проделать шаги a-d для всех терминов

6. Когда все элементы в тексте будут помечены, создайте список терминов: Ссылка \ Предметный указатель.

14. Создайте новый раздел в конце документа и вставьте Список рисунков (Ссылки \ Список иллюстраций).

15.  Создайте новый раздел в конце документа и вставьте Список литературы.

Задайте для перечня литературы нумерованный список. В формате абзаца поменяйте Выступ первой строки на Отступ 

16.  Создайте Титульную страницу для курсового проекта (см Образец).

17.  Создайте Верхний колонтитул (см. образец):

Для вставки названия текущего раздела в колонтитул:

Конструктор \ Вставка \ Экспресс-блоки \ Поле...
Выбрать поле StyleRef. В колонке Свойства поля укажите Имя стиля Заголовок 1

Обратите внимание, что титульная страница не должна содержать колонтитулы, страницы «Содержание», «Список терминов», «Список иллюстраций» и «Список Литературы» содержат только нижнюю границу и номер страницы.

18.  На заключительном этапе пометьте документ как окончательный (Файл \ Сведения \ Защитить документ \ Пометить как окончательный).

19.  Сохраните документ под названием Курсовой проект Ваша Фамилия.docx.
